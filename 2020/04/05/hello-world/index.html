感谢codesheep的教程

以下所有过程在Windows7 64进行演示

NO.1 下载并安装Node.js

进入Node.js官网

下载完成后进行安装，一直Nest下一步就好.

NO.2 下载并安装git

进入git官网
找到Downloads.

进入下载界面，选择对应系统版本下载，这里用Windows系统演示所以下载Windows版本.

选择对应的版本进行下载.

下载完进行安装，安装方法与安装Node.js安装方法相同，一直下一步就好.

NO.2.5 git的用户配置

使用git进行安装前应先设置git的用户名以及邮箱，这一点很重要，因为每一个 Git 提交都会使用这些信息，它们会写入到你的每一次提交中，不可更改：

$ git config –global user.name “Your Name”
$ git config –global user.email youremail@example.com

如果不进行用户信息配置，那么博客配置到github的数据库时会出现问题（亲测，说多了都是泪）.

NO.3 安装Hexo

tips：这里安装和后续操作可以使用Windows自带的cmd进行安装，但推荐使用git bash；以下用git bash进行演示.
安装前我们要新建文件夹来存放我们的Hexo文件，这里以文件blog为例

打开blog，右键选择–git bash here–

node -v 查看node版本

上面命令一般用于检测是否安装成功（显示版本号即安装成功）
为了以后下载的速度，我们要先下载cnmp.

npm install -g cnpm –registry=http://registry.npm.taobao.org

这行命令是安装淘宝的cnmp管理器

cnpm -v 查看cnmp版本

OK，目前为止前戏都已经准备好，接下来让我们安装hexo框架

cnpm install -g hexo-cli
hexo -v 查看hexo版本

NO.4启用hexo

接下来确认在blog文件夹内

hexo init 生成初始化博客

初始化完成后

hexo s 启动本地博客服务
http://localhost:4000/ 用浏览器输入网址即可打开本地博客
